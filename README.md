# 專案 (vue + nodejs-express)
https://github.com/ayay459547/Cfrontend.git

## Project setup
```
cd backend
npm install
cd ..

cd frontend
npm install
cd ..
```

### Compiles and hot-reloads for development
```
cd backend
npm run dev
cd ..

cd frontend
npm run serve
cd ..
```

### Compiles and minifies for production
```
cd frontend
npm run build
```
